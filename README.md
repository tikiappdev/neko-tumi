# ねこ積みタワー

左右に動くねこをタップで落とし、重なった部分だけを積み上げる、短時間で遊べるブラウザゲームです。日本語UI、スマートフォン縦画面を中心に、PC・タブレット・キーボード操作にも対応しています。

## 遊び方とルール

1. 「あそぶ！」を押します。
2. ねこが下のねこの上に来たら、盤面をタップまたはクリックします。キーボードでは `Space` / `Enter` でも落とせます。
3. 重なった部分だけが残り、はみ出した分だけ次のねこが小さくなります。
4. ほぼ同じ位置なら「ぴったり！」になり、幅を保ったままボーナス得点とコンボを獲得します。
5. まったく重ならなければ終了です。「もういっかい！」ですぐ再挑戦できます。

成功は1段100点。ぴったりはさらに100点、連続ぴったりには最大250点の追加ボーナスがあります。段数に応じて速度と背景（地上、町、雲、夕焼け、夜空、宇宙）が変わります。

## ねこと解放要素

しろねこ、くろねこ、みけねこ、とらねこ、はちわれ、ねむねこ、ごきげんねこ、きんのねこ、にじいろねこの9種類を実装しています。最高段数、総プレイ回数、ベストスコア、ぴったりコンボの条件を達成すると新しい柄が増えます。柄による性能差はありません。

## 技術構成

- React + TypeScript + Vite
- 通常CSS（外部画像・CDNなしのインラインSVG猫）
- Vitest + React Testing Library
- Web Audio APIによる軽量な効果音
- localStorageによる記録保存

ルール判定は `src/game` の純粋関数、ゲーム進行は `src/game/useGame.ts`、表示は `src/components`、保存処理は `src/storage/playerData.ts` に分離しています。

## ローカル起動

Node.js 20以降を推奨します。

```bash
npm install
npm run dev
```

表示されたローカルURLをブラウザで開いてください。

## 品質確認

```bash
npm run lint
npm run typecheck
npm test
npm run build
```

テストでは、初期状態、左右移動と端反射、重なり・ぴったり・完全な外れ、スコアとコンボ、背景・解放条件、ベスト記録、保存・復元、壊れた保存データ、音設定、主要画面遷移を確認します。

## 本番ビルド

```bash
npm run build
```

成果物は `dist/` に生成されます。

## Vercelで公開

GitHubへリポジトリを登録し、VercelでImportします。

- Framework Preset: `Vite`
- Install Command: `npm install`
- Build Command: `npm run build`
- Output Directory: `dist`

`vercel.json` にSPA向けrewriteを設定済みです。外部API、ログイン、サーバー、データベース、環境変数は不要です。

## 保存データ

localStorageの `neko-tsumi-tower:player-data` に、スキーマバージョン、ベストスコア、最高段数、総プレイ回数、最大ぴったりコンボ、音のON/OFF、解放済みねこIDを保存します。JSON破損、型不正、読み書き例外では安全な初期値を使い、ゲームを継続します。

ブラウザのサイトデータを削除すると記録は初期化されます。個人情報は保存・送信しません。
